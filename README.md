# 📊 Sales Dashboard Project

## 📌 Overview

This project focuses on analyzing sales data to generate meaningful business insights using **Python, SQL, and Power BI**.
It demonstrates end-to-end data analysis — from data cleaning to interactive dashboard creation.

---

## 🛠️ Tools & Technologies

* 🐍 Python (Pandas)
* 🗄️ MySQL
* 📊 Power BI

---

## 📂 Dataset

The dataset contains:

* Customer information
* Product details
* Sales, profit, and quantity
* Order and shipping dates

---

## 🔧 Data Processing

* Cleaned missing and inconsistent data using Pandas
* Converted date columns and extracted year/month/day
* Created new features:

  * Total Amount
  * Delivery Days
  * Order Frequency

---

## 📊 Dashboard Features

### 🔹 KPI Metrics

* Total Sales
* Total Orders
* Total Customers
* Average Order Value (AOV)

### 🔹 Visualizations

* Sales by Customer Segment
* Sales Trend (Time-based analysis)
* Top 10 Products by Sales
* Sales by Sub-Category
* Profit by Category

### 🔹 Filters (Slicers)

* Month
* Region
* Product Category

---

## 📈 Key Insights

* Corporate segment contributes highest sales
* Office Supplies category generates significant profit
* A small number of products drive majority revenue
* Sales vary across regions and months

---

## 🧠 Customer Segmentation (RFM Analysis)

* Performed RFM (Recency, Frequency, Monetary) analysis
* Segmented customers based on purchasing behavior
